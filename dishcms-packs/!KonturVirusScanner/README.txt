1) Работает только на UNIX системах
2) После заверешения работы не забудьте УДАЛИТЬ сам файл сканирования и логи ksort_*
3) Правила, на данный момент, задаются только в методе KSort::rules()


 Краткая инструкция:
- можно отметить несколько файлов "Проверено" и нажать "Сохранить" для тех, которые не являются вирусными http://joxi.ru/82QVJlYSw7qz9A 
(используется для удобства восприятия таблицы с файлами)
- можно отметить несколько файлов "Удалить" и нажать "Удалить", соответственно для удаления http://joxi.ru/KAxe0qzcZDBEXr
- Форма "Маска" используется для массового выделения файлов, где в поле указывается js REGEXP выражение по именам файлов. 
Кнопки рядом "Сохранить", "Удалить" и "Сбросить" используются только для выставления галочек. http://joxi.ru/BA0dqlpSMYKbDA
- Кнопка "Карантин" http://joxi.ru/n2Y8pokSbad1jm используется для перемещения файлов в карантин. 
Фактически происходит простое добавление к имени в файла постфикса .ksuspected
- правка файлов происходит в кодировке по умолчанию (может не соответствовать UTF-8). При необходимости правки, рекомендуется проверить на одном из файлов.
