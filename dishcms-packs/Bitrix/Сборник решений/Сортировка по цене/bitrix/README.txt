Добавление сортировки товара по цене

1) Добавить в инфоблок каталога два свойства (для товара)
Название / Тип / Сорт. / Код
СИСТЕМНЫЙ [Минимальная Цена] / Число / 9000 / MINIMUM_PRICE
СИСТЕМНЫЙ [Максимальная Цена] / Число / 9000 / MAXIMUM_PRICE
    
2) Скопировать код из "php_interface/init.php" в соответствующий файл "bitrix/php_interface/init.php"
3) Скопировать папку "kontur" в "bitrix/php_interface"
4) В нужном месте шаблона вставить <? KonturShowSortByPrice(); ?>
5) Изменить настройки компонента bitrix:catalog
	5.1) / Настройки списка / По какому полю сортируем товары в разделе: (другое) / ={$_REQUEST["sort"]?:"CATALOG_AVAILABLE"}
	5.2) / Настройки списка / Порядок сортировки товаров в разделе: (другое) / ={$_REQUEST["method"]?:"desc"}
